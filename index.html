DOM Javascript

document

Sirve al javascript que haya sido vinculado con el HTML, pueda dar contenido de
texto y clases. Sin ningun otro atributo, dentro del selector. Solo se declarardo como
variable y de allí añadir sus propiedades.

querySelector

Para que document haga efecto en el HTML, se necesita
de un querySelector para dar ese efecto hacia el HTML.

querySelectorAll

Como su mismo nombre lo dice, sirve para seleccionar varios selectores
en donde se devolvera como un array. Y se podra dar modificación con ello.

InnerHTML

Sirve para en codificar la sintaxis, una vez declarado
el querySelector,

TextContent

Parecido al InnerHTML, solo que se agregan solo cadena de texto.

ClassList.add

Se agrega un clase para, funcionar indirectamente,
con el CSS.

getElementId

Muy parecido al querySelector, solo que ahora va para el id de
un selector en especifíco que lleva el mismo nombre del atributo.
